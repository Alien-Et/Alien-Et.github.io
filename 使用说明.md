# Hexo网站使用说明

## 一、文章添加方法

### 1. 创建新文章

方法一：使用Hexo命令
```bash
# 在项目根目录执行
npx hexo new "文章标题"
```

方法二：手动创建文件
- 在 `source/_posts/` 目录下创建 `.md` 文件
- 文件名建议使用英文或拼音，如 `solar-panel-launch.md`
- 文件开头添加Front Matter（前置数据）

### 2. 文章格式

每个文章文件开头需要添加以下Front Matter：

```yaml
---
title: 文章标题
date: 2024-12-01 00:00:00
categories: [分类1, 分类2]
tags: [标签1, 标签2]
description: 文章描述
---

文章内容...
```

### 3. 多语言文章

- 英文文章：放在 `source/en/_posts/` 目录
- 繁体中文文章：放在 `source/zh-TW/_posts/` 目录
- 日语文章：放在 `source/ja/_posts/` 目录

## 二、网站图标素材修改

### 1. 图片文件位置

- 网站图标：`themes/enterprise/source/images/` 目录
- 文章图片：`source/images/` 目录
- 主题图片：`themes/enterprise/source/images/` 目录

### 2. 主要图片文件

| 图片类型 | 位置 | 说明 |
|---------|------|------|
| 网站图标 | `themes/enterprise/source/images/hero-bg.svg` | 网站首页背景图 |
| 团队头像 | `themes/enterprise/source/images/about/team*.svg` | 团队成员头像 |
| 合作伙伴 | `themes/enterprise/source/images/partners/partner*.svg` | 合作伙伴Logo |
| 资质证书 | `themes/enterprise/source/images/about/cert*.svg` | 公司资质证书 |
| 公司图片 | `themes/enterprise/source/images/about/company.svg` | 公司外观图片 |
| 文章图片 | `themes/enterprise/source/images/posts/*.svg` | 文章相关图片 |

### 3. 修改方法

1. 替换图片文件：直接替换对应目录下的图片文件
2. 保持文件名一致：确保替换后的文件名与原文件名相同
3. 使用SVG格式：推荐使用SVG格式图片，确保清晰度和加载速度
4. 图片尺寸：保持与原图片相同的尺寸比例

## 三、网站配置修改

### 1. 基本配置

编辑 `_config.yml` 文件：

```yaml
# 网站信息
title: 网站标题
description: 网站描述
keywords: 关键词
author: 作者名

# 部署配置
deploy:
  type: git
  repo: https://github.com/Alien-Et/Alien-Et.github.io.git
  branch: gh-pages
```

### 2. 多语言配置

编辑对应语言文件：
- 简体中文：`themes/enterprise/languages/zh-CN.yml`
- 英文：`themes/enterprise/languages/en.yml`
- 繁体中文：`themes/enterprise/languages/zh-TW.yml`
- 日语：`themes/enterprise/languages/ja.yml`

### 3. 主题配置

编辑 `themes/enterprise/_config.yml` 文件：

```yaml
# 主题配置
menu:
  home: /
  about: /about/
  products: /products/
  news: /news/
  contact: /contact/
```

## 四、部署网站

### 1. 构建网站

```bash
# 清理缓存
npx hexo clean

# 构建网站
npx hexo generate
```

### 2. 部署到GitHub Pages

```bash
# 部署网站
npx hexo deploy
```

### 3. 启用GitHub Pages

1. 登录GitHub，进入 `Alien-Et/Alien-Et.github.io` 仓库
2. 点击 "Settings" → "Pages"
3. 在 "Build and deployment" 部分：
   - Source 选择 "Deploy from a branch"
   - Branch 选择 "gh-pages"
   - 点击 "Save"
4. 等待部署完成后，访问 `https://alien-et.github.io`

## 五、常见问题

### 1. 文章不显示
- 检查文章是否在正确的目录中
- 检查Front Matter格式是否正确
- 执行 `npx hexo clean && npx hexo generate` 重新构建

### 2. 图片不显示
- 检查图片路径是否正确
- 确保图片文件存在
- 执行 `npx hexo clean && npx hexo generate` 重新构建

### 3. 部署失败
- 检查GitHub仓库地址是否正确
- 确保GitHub仓库存在且可访问
- 检查网络连接是否正常

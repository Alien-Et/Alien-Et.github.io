为什么选择户用光伏

户用光伏系统具有投资门槛低、收益稳定、环保节能等优势，越来越多的家庭选择在自家屋顶安装光伏电站。

主要优势

  • 稳定收益：发电收益持续20年以上
  • 投资回报：投资回收期5-7年
  • 节能减排：每kW年减排约1吨CO₂
  • 屋顶保护：延长屋顶使用寿命
  • 政策支持：国家鼓励分布式光伏发展

安装前准备

1. 屋顶条件评估

适合安装的条件

  • 屋顶面积：每kW约需7-10平方米
  • 屋顶朝向：朝南最佳，东南、西南次之
  • 屋顶倾角：15-35度为宜
  • 承重能力：≥30kg/平方米
  • 使用年限：预计使用15年以上

不适合安装的情况

  • 屋顶严重老化或损坏
  • 周边有高大建筑遮挡
  • 屋顶产权不清晰
  • 违章建筑或临时建筑

2. 用电情况分析

了解家庭用电情况：

用电类型 日均用电量 建议装机容量
低用电家庭 <10kWh 3-5kW
中等用电家庭 10-20kWh 5-10kW
高用电家庭 >20kWh 10-20kW

3. 资金准备

投资预算参考

  • 设备成本：约3-4元/W
  • 安装费用：约0.3-0.5元/W
  • 其他费用：约0.2元/W
  • 总投资：约3.5-4.5元/W

融资方式

  • 全款购买：收益最高
  • 银行贷款：首付30%左右
  • 租赁模式：零首付，收益分成
  • 企业投资：屋顶出租收取租金

设备选型指南

1. 光伏组件

选型要点

  • 品牌选择：一线品牌优先
  • 功率选择：根据屋顶面积确定
  • 类型选择：单晶硅效率更高
  • 质保期限：25年线性功率质保

XX科技推荐产品

型号 功率 效率 尺寸 适用场景
G-450W 450W 21.0% 2100×1050mm 小型屋顶
G-550W 550W 21.8% 2278×1134mm 中型屋顶
G-600W 600W 22.5% 2384×1303mm 大型屋顶

2. 逆变器

选型要点

  • 容量匹配：逆变器容量≥组件容量的1.1倍
  • 品牌选择：知名品牌，售后服务好
  • 功能选择：支持智能监控
  • 防护等级：IP65以上

类型对比

类型 优点 缺点 适用场景
组串式 安装灵活、效率高 单点故障影响大 大多数户用
微逆 组件级优化 成本较高 复杂屋顶
优化器 组件级监控 需配合组串式 有遮挡场景

3. 支架系统

选型要点

  • 材质：铝合金或不锈钢
  • 抗风能力：≥12级台风
  • 防腐处理：阳极氧化或镀锌
  • 安装方式：不破坏屋顶防水

4. 电缆与配电

电缆选型

  • 直流电缆：PV专用电缆，4mm²或6mm²
  • 交流电缆：YJV电缆，根据电流选择截面
  • 接地电缆：黄绿双色电缆

配电设备

  • 直流汇流箱（多路组串需要）
  • 交流配电箱
  • 断路器、浪涌保护器
  • 电能表

安装流程

第一步：项目备案

所需材料

  • 身份证复印件
  • 房产证或宅基地证明
  • 物业同意书（小区需要）
  • 屋顶产权承诺书

办理流程

  1. 向当地发改委或能源局提交备案申请
  2. 等待审核，一般5-10个工作日
  3. 获取备案证明文件

第二步：并网申请

所需材料

  • 备案证明
  • 身份证复印件
  • 房产证明
  • 光伏系统设计方案
  • 设备认证证书

办理流程

  1. 向当地供电公司提交并网申请
  2. 供电公司现场勘察
  3. 出具接入方案
  4. 签订并网协议

第三步：设备安装

安装步骤

  1. 支架安装

    • 测量放线定位
    • 安装底座固定
    • 安装导轨横梁
    • 调整水平角度

  2. 组件安装

    • 搬运组件到屋顶
    • 按顺序铺设组件
    • 固定组件压块
    • 连接组件电缆

  3. 逆变器安装

    • 选择安装位置
    • 固定逆变器
    • 连接直流电缆
    • 连接交流电缆

  4. 配电安装

    • 安装配电箱
    • 连接各路电缆
    • 安装保护器件
    • 接地系统施工

第四步：并网验收

验收流程

  1. 安装完成后申请验收
  2. 供电公司现场验收
  3. 安装双向电表
  4. 签订购售电合同
  5. 正式并网运行

运维保养

日常维护

  • 定期清洁组件表面
  • 检查设备运行状态
  • 关注发电量变化
  • 及时处理异常情况

注意事项

  • 禁止踩踏组件
  • 雷雨天气注意安全
  • 发现故障及时报修
  • 保留运维记录

常见问题

Q1：阴雨天能发电吗？

可以发电，但发电量会降低。阴天约为晴天的20-30%，雨天约为晴天的5-10%。

Q2：组件需要经常清洗吗？

一般2-3个月清洗一次即可。如果当地灰尘较多，可适当增加清洗频率。

Q3：光伏系统有辐射吗？

光伏系统产生的是直流电，没有电磁辐射，对人体无害。

Q4：冬天发电量会下降吗？

冬季日照时间短，发电量会有所下降，但低温有利于组件发电，整体影响不大。
